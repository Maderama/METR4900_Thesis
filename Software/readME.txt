The following files contain the software implementation of the Growroom Thesis project by Madison Beare. 

theMonarch gets flashed to the arm/theMonarch PCB. 

The wemosTransmitter code gets flashed to the turntable PCB. 

To run the website, theMonarchWebsite.py will need to be modified to include the correct IP address. To run the website from the command line, a virtual workpspace was set up, and the following command run: 
python theMonarchWebsite.py. 

This assumes the correct directory has been traversed to.
